## Master Language Analysis

This project analyzes the language used in works written by a selected Master. 

## Command-line arguments 
-- dictionary : path to the dictionary file

-- works : comma-separated list of Master's works

-- dictionary stats : dictionary statistics 

-- work-stats : statistics of the Master's works

-- no-words : displays words not found in dictionary

--frequencies : displays the n most frequent words in each work

--output : path to the output text file

## Output

- a text file containing statistics results 

- plots presenting the similarity between the analyzed works

- similarity.csv containing similarity scores between analyzed works
